WoSo Stats

This repository is for a collection of stats from women's soccer matches

The WoSo Stats Shiny app is currently the best place to browse and analyze match stats, at https://amj2012.shinyapps.io/wosostats/. This app is recommended, at the R Markdown documents shown below are only for a few matches.

The analysis folder is where to find the documents that show the results of each match analysis. Working on putting these into a blog to make it easier to read. Soon.

In the meanwhile, here is a list of the individual match analyses I actually have been able to do (links to code and source data are provided in each one):

  • NWSL 2015
    • Portland Thorns FC vs. Seattle Reign FC - July 22, 2015
    • NWSL 2015 Semifinal - Chicago Red Stars vs. FC Kansas City
    • NWSL 2015 Semifinal - Seattle Reign FC vs. Washington Spirit
    • NWSL 2015 Championship - Seattle Reign FC vs. FC Kansas City
  • D1 Feminine 2015/2016
    • Paris-Saint Germain vs. Lyon - February 5, 2016
  • 2016 Olympics CONCACAF Qualifying Tournament
    • Group Match: USA vs. Costa Rica - February 10, 2016
    • Final: USA vs. Canada - February 21, 2016
  • 2016 International Friendlies
    • USA vs. Ireland - January 23, 2016

The code folder is where to find the R code that is used to process and analyze the source Excel and csv files.

The resources folder is where to find necessary and helpful resources for how to log stats on your own (please help! volunteers needed!) and definitions for what is being measured and analyzed.

The source folder is where to find the Excel and csv files of stats logged for each match.
